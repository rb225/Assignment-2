Purpose

The purpose of this website is to learn the Linux,vi,GIT commands and get the practical hands-on experience in implementing the first website using the github commands.I used basic linux commands to change directories and creating new files.I also used VI editor to create the HTML page.Later I also used git through Git Bash and also used Git Commands to perform the operations


Linux Commands

  1. ls: Directory listing
  2. pwd: Showing Current Directory
  3. cd dir:Change Directory
  4. touch: Create or update file
  5. ps: Display all current processes

VI Commands

  1. vi: Edit Filename starting at line 1
  2. wq:quit vi, writing out modified file to file named in original invocation
  3. q:quit (or exit) vi

GIT Commands

  1. git status:show modified files in working directory, staged for your next commit
  2. git add (file):add a file as it looks now to your next commit (stage)
  3. git config --global user.name “[firstname lastname]”:set a name that is identifiable
  4. git config --global user.email “[valid-email]”:set an email address that will be associated with each history marker
  5. git init: Initializing the current Directory into GIT working Directory
